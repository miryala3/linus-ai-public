## What it does

Daily planning app that runs entirely on your device. No subscription, no data in the cloud, no account required.

- Morning briefing — surfaces priorities, appointments, and overdue tasks
- AI scheduling — plans your day based on energy levels and deadlines
- Focus coach — Pomodoro timer with AI-generated task breakdowns
- Habit tracker — streak tracking with AI pattern recognition
- End-of-day reflection — guided journaling and tomorrow planning
- Connects to LINUS-AI on port 9480 for deeper AI capabilities
- Uses a `LNDY-` license key

## Install

**macOS** — open DMG → drag to Applications → right-click → Open on first launch  
**Linux (.deb)** — `sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 && sudo dpkg -i linus-day-*.deb`  
**Linux (AppImage)** — `chmod +x linus-day-*.AppImage && ./linus-day-*.AppImage`
