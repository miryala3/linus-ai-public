## What it does

Adds an AI assistant sidebar inside QuickBooks, Xero, Stripe, HubSpot, Salesforce, Gmail, and Outlook. Connects to LINUS-AI on port 9480 and LINUS BizCore for context-aware suggestions.

- Drafts invoices, replies, and reports directly inside your existing tools
- Summarises open tasks, outstanding invoices, and customer history
- Works entirely offline — no data sent to external servers
- Uses a `LNBC-` key (same as BizCore desktop app)

**Included free** with any BizCore Starter, Business, or Unlimited license.  
Available standalone for $99 one-time.

## Install

**Chrome / Edge / Brave / Opera**  
1. Unzip the `.zip` file  
2. Go to `chrome://extensions` → enable Developer mode  
3. Click "Load unpacked" → select the unzipped folder

**Firefox**  
`about:addons` → gear icon → Install Add-on From File → select `.xpi`

**After install:** Extension → Settings → License Key → enter `LNBC-` key → Save
