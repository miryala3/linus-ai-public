## What it does

Desktop travel workspace that stores all your trip data locally. Plan, track, and review trips without sending your itinerary to cloud servers.

- Itinerary builder with AI-assisted planning via LINUS-AI
- Hotel and flight search with price tracking (offline-first, caches results)
- Expense tracking with category tagging and currency conversion
- Document vault — passports, visas, confirmations, all encrypted locally
- Packing list builder with AI suggestions based on destination and dates
- Uses a `LNTR-` license key

## Install

**macOS** — open DMG → drag to Applications → right-click → Open on first launch  
**Linux (.deb)** — `sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 && sudo dpkg -i trips-*.deb`  
**Linux (AppImage)** — `chmod +x trips-*.AppImage && ./trips-*.AppImage`
