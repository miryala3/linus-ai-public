## What it does

Complete reservation management system for restaurants, hotels, and event venues. Runs on your hardware — flat license, no commission, no cloud dependency.

- Reservation engine with floor plan management and table assignments
- AI yield optimiser — maximises covers and revenue per session
- No-show prediction with automated waitlist promotion
- POS integration — syncs orders, payments, and covers in real time
- AI booking agent — handles online reservation requests autonomously
- Guest intelligence — dietary notes, visit history, preferences
- Uses a `LNRV-` license key

## Install

**macOS** — open DMG → drag to Applications → right-click → Open on first launch  
**Linux (.deb)** — `sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 && sudo dpkg -i reserve-*.deb`  
**Linux (AppImage)** — `chmod +x reserve-*.AppImage && ./reserve-*.AppImage`
