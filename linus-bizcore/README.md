## Install

**macOS**
```bash
xattr -d com.apple.quarantine bizcore-LINUS-BizCore_4.0.1_aarch64.dmg
# Open DMG → drag to Applications → right-click → Open on first launch
```

**Linux (.deb)**
```bash
sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 libayatana-appindicator3-1 librsvg2-2
sudo dpkg -i bizcore-LINUS-BizCore_4.0.1_amd64.deb
```

**Linux (AppImage)**
```bash
chmod +x bizcore-LINUS-BizCore_4.0.1_amd64.AppImage
./bizcore-LINUS-BizCore_4.0.1_amd64.AppImage
```
