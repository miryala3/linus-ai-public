## What it does

Desktop studio for designing and distributing products on the LINUS-AI mesh network. Build peer-to-peer workflows with offline-first storage and AI-assisted design.

- Visual workflow builder — design partner, procurement, and service-network flows
- Tender and RFP management with AI scoring
- Peer distribution — publish apps directly to connected nodes
- Offline-first storage with sync on reconnect
- Integrates with LINUS-AI on port 9480 for AI-assisted design
- Resilience planning and business continuity workflows
- Uses a `LNFG-` license key

## Install

**macOS** — open DMG → drag to Applications → right-click → Open on first launch  
**Linux (.deb)** — `sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 && sudo dpkg -i forge-*.deb`  
**Linux (AppImage)** — `chmod +x forge-*.AppImage && ./forge-*.AppImage`
