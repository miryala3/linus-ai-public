### Translation / transliteration statement

### Likelihood-of-confusion search summary

Searches conducted against TESS (USPTO TM database) and EUIPO eSearch Plus
for "LINUS", "LINUS AI", "LINUSAI" on 2026-05-13:

- `LINUS` (live registration #2,367,xxx, owner Linus Pauling Institute) —
  used for educational/scientific research services in Class 041; **no
  conflict** with our 009/042 software claim.
- `LINUS LEARNING` (live #5,...) — academic/textbook publishing in Class 016;

- `LINUX` (live, Linux Foundation) — operating system software in Class 009;

**Counsel notes**: Confirm Linux Foundation's stance on marks beginning with

the mark is "manifestly distinct" and goods don't compete. We are
non-competing (we're software *built on top of* Linux, not an OS). Worst case
the USPTO examiner objects on §2(d) and we respond with a coexistence
agreement.

## Common drafting notes (apply to all three)

### Specimen of use templates

For Class 009 (downloadable software), USPTO accepts:

1. Screenshot of the product **download page** showing the mark adjacent
   to a download button.
2. Screenshot of the **installed application's About panel** showing the
   mark.
3. Screenshot of the **app icon and app name** as shown in the OS launcher.

For Class 042 (SaaS), USPTO accepts:

1. Screenshot of the **logged-in service** showing the mark in the page
   header (e.g., hub.linus-ai.com top-left "LINUS-AI" logo).
2. Screenshot of the **service description page** showing the mark.
3. Marketing material showing the mark used in conjunction with the
   service.

### Filing fees (TEAS Plus, 2026 schedule — confirm current rates)

- TEAS Plus: $250 per class per mark
- Statement of Use (if filing §1(b) intent-to-use): $100 per class
- Allege Use (post-publication): $100 per class

For three marks × two classes each at TEAS Plus: **$1,500 USPTO fees**.

### Timeline expectations

- File → first office action: ~6 months
- Publication for opposition: ~9 months
- 30-day opposition window
- Registration certificate: ~12-14 months total

### Maintenance

- §8 affidavit: due between 5th and 6th anniversary of registration. $325.
- §9 renewal: due between 9th and 10th anniversary. $425 per class.
- §15 incontestability (optional): file with the §8 affidavit. $250.
