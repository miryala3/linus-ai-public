## What it does

Browser extension that connects to your on-premise aiMED Medical Server and adds clinical AI directly into your workflow — inside QuickBooks, Xero, EHR portals, and more.

**8 workflow tabs:**
- Chart Summary · Pre-Charter · Inbox Zero · Billing Optimizer
- Patient Educator · Quick Tools · Insurance Verify · SuggestCare

Uses a `LNAM-` license key. Clinic and Hospital licenses of aiMED Medical include an Extension license.

## Install

**Chrome / Edge / Brave / Opera**  
1. Unzip the `.zip` file  
2. Go to `chrome://extensions` → enable Developer mode  
3. Click "Load unpacked" → select the unzipped folder

**Firefox**  
`about:addons` → gear icon → Install Add-on From File → select `.xpi`

**After install:** Extension → Settings → Account → enter your `LNAM-` license key → Save  
Verify aiMED Medical Server is running at `http://localhost:9500` before using.
