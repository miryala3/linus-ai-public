## What it does

Desktop app for professional networking built on encrypted peer-to-peer connections. Your contacts, messages, and portfolio live on your machine — not on a platform's servers.

- Verified identity (KYC + KYB) and trusted introductions
- Direct encrypted messaging — no relay servers reading your DMs
- Project boards, portfolio hosting, and skill endorsements — all local
- Connects to LINUS-AI for AI-assisted profile writing and outreach drafts
- AI network discovery — finds relevant peers based on skills and goals
- Uses a `LNCX-` license key

## Install

**macOS** — open DMG → drag to Applications → right-click → Open on first launch  
**Linux (.deb)** — `sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 && sudo dpkg -i connectx-*.deb`  
**Linux (AppImage)** — `chmod +x connectx-*.AppImage && ./connectx-*.AppImage`
