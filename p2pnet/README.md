## What it does

Private mesh infrastructure for teams and businesses. No central servers, no cloud relay — all routing is peer-to-peer.

- Verified identity (KYC + KYB) and trusted peer introductions
- AI-powered network discovery and market entry planning
- DePIN compute: Akash (CPU) + io.net (GPU) + Filecoin (storage)
- Blockchain trust — Polygon hash anchoring via Gelato Relay (no gas headaches)
- Zero-commission payments: Dwolla ACH ($0.50 flat) + GoCardless SEPA (€0.20 flat)
- REST API on `:7480` alongside the native GUI
- mDNS auto-discovery, private DNS, NAT traversal
- All features work offline without API keys — configure via environment variables to activate live providers
- Uses a `LNPN-` license key

## Install

**macOS** — open DMG → drag to Applications → right-click → Open on first launch  
**Linux (.deb)** — `sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 && sudo dpkg -i p2pnet-*.deb`  
**Linux (AppImage)** — `chmod +x p2pnet-*.AppImage && ./p2pnet-*.AppImage`

REST API available at `http://localhost:7480` after launch.
