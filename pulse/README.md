## What it does

A suite of personal AI mini-apps that run on your machine. All data stays local — no telemetry, no subscriptions required.

- **AI Chat Companion** — private conversations with local LLMs via LINUS-AI
- **Journal** — daily entries with AI sentiment tagging and pattern recognition
- **Goal Deconstructor** — breaks goals into actionable steps with progress tracking
- **Financial Categoriser** — imports transactions and categorises them with AI
- **Summariser** — paste any text, document, or URL for instant summary
- Uses a `LNPA-` license key

## Install

**macOS** — open DMG → drag to Applications → right-click → Open on first launch  
**Linux (.deb)** — `sudo apt-get install -y libwebkit2gtk-4.1-0 libgtk-3-0 && sudo dpkg -i pulse-*.deb`  
**Linux (AppImage)** — `chmod +x pulse-*.AppImage && ./pulse-*.AppImage`
